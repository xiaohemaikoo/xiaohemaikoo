Natural language Processing

Language Evolution

Research on the cross-language lexical semantic changes, design an unsupervised clustering model applied to mBERT word embeddings to generate temporal-spatial dynamics of lexical semantic changes.

The proposed method is able to capture nuanced changes in both high- and low-frequency word senses across time and languages, including the acquisition and loss of these senses over time. This method achieves the state of art result in the SemEval2020 binary classification task across four languages. It is also a visualization tool to detect semantic changes in both intra-language and inter-language setups.

This work is published in the conference EACL2024. The code and data are made publicly available.

Novel word senses detection

Design a lexical semantic change detection model to detect novel senses not present in the dictionary. Utilize the capabilities of large language models to summarize the detected novel senses and generate dictionary definitions.

In Subtask 1, the proposed system outperforms the baseline system by a large margin.

In Subtask 2, the system ranks first in Finnish and German, and ranks second in Russian on the test-phase leaderboard.

This work is published at the 5th International Workshop on Computational Approaches to Historical Language Change 2024 (LChange24). The code and data are made publicly available.
